# Healthcare Patient Visit Analysis

## Project Overview

This project analyzes 15 years of hospital encounter data to identify monthly trends, patient growth, service utilization patterns, and frequent medical conditions.

The objective is to generate operational insights from structured healthcare data using SQL and Python.

## Objectives

- Analyze monthly hospital visit trends
- Measure unique patient growth over time
- Identify the busiest service types
- Detect the most frequent medical conditions
- Evaluate condition distribution across service types
